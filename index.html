NOTES

The main research product is an AXIO/1 formal runtime that is able to run simple typed lambda programs on verifiable interpreter. But nobody can limit one developing their own higher languages with extraction to that interpreter. In order to keep that process in one place under cathedral type of open source software management proceess — the AXIO/1 appears.

Interaction Networks Optimal Lambda Interpreter.

Joe

Joe/ARM-9.5 is a CoqAsm verified interpreter based on Interaction Networks of Lambda Calculus, the core of LIB/IO foundation and libraries (see Kind, Kindelia language and libraries). In addition to Joe we are using sub-optimal BEAM Ericsson production JIT-interpreter for ERP/N2O/IO foundation and libraries.

Sound and Consistent Predicative Formal Languages.

Alonzo

Alonzo/OCaml is an STLC type system.

Henk

Henk/Erlang is a Pure Type System in the style of Coquand/Huet with infinite numbere of universes.

Per

Per/Elixir is a Pi, Sigma, Id prover with Calculus of Inductive Constructions.

Anders

Anders/OCaml is a Homotopy Type System (Voevodsky) with Strict Equality and CCHM Agda primitives.

Urs

The Urs/OCaml is an equivariant superhomotopy type system with fermion and boson modalities built-in into type checker.
